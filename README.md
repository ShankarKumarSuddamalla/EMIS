# EMIS
Education Management Information System



To Run this Project:
1)git clone https://github.com/shankar-8374/EMIS.git
2)npm install
3)open 3 terminals
   in 1 terminal:1)cd client
                 2)npm start
   in 2nd terminal:1)cd server
                   2)nodemon index.js
  in 3rd terminal:1)cd chatcord
                  2)nodemon server.js
